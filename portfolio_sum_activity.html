  2. Calculate Total Points for Each Student using SUM formulas
    =SUM(B2:F2)

    Apply this formula to each student's row to total all their assignment scores

  3. Calculate Assignment Totals for the entire class
    =SUM(B2:B26)

    Create totals for each assignment column (Quiz 1, Quiz 2, Midterm, Final, Project)

  4. Create Class Statistics using SUM formulas
    =SUM(G2:G26)

    Calculate total points earned by all students combined
