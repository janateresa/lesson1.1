VLOOKUP Quick Reference
Syntax
=VLOOKUP(lookup_value, table_array, column_index, [range_lookup])
  • lookup_value:
    What you're searching for (student ID, course code, etc.)
  • table_array:
    The data range where Excel should search (A1:D100)
  • column_index:
    Which column contains the information you want back (2 for column B, 3 for column C, etc.)
  • range_lookup:
    Use FALSE for exact matches (recommended for educational data)
Scenario 1: Student Information Lookup
You're managing a student database where column A contains Student IDs, column B has student names, column C has their majors, and column D has their email addresses. You need to quickly find student information when you only have their ID number.
Example
=VLOOKUP("STU-2024-001", A2:D100, 3, FALSE)
Scenario 2: Course Catalog Integration
Your course catalog has course codes in column A, course titles in column B, and instructor names in column C. You want to automatically populate course titles when faculty enter course codes.
Example
=VLOOKUP(B2, CourseList!A:C, 2, FALSE)
